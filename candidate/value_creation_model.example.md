# Value Creation Model — TEMPLATE

Copy this file to `candidate/value_creation_model.md` (gitignored) and replace the
prompts below with your own words. This is **optional** — the pipeline runs fine
without it. When present, it helps the tailoring articulate how you authentically
create value, instead of just matching keywords.

It describes **recurring patterns** — how you create value across roles, independent
of job titles, industries, or specific achievements. It is **not** a list of
accomplishments (those live in your CV) and it is **never used to invent claims**: the
tailoring only uses it to choose emphasis and framing for things already in your CV.

Keep it concrete and personal. A few sentences per section is plenty.
